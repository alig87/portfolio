About

The developer behind the code

Introduction

I'm a Flutter mobile developer focused on building complete, production-ready applications — not just UI screens. My work spans the full stack of mobile development: Flutter/Dart on the front end, native Android (Kotlin) for system-level features like Accessibility Services and device-level locks, and REST API integrations on the backend side.

I care about architecture as much as appearance. Every app I build follows a structured pattern — MVVM, Clean Architecture, or BLoC — because code that's easy to extend today is the same code that won't break in production six months from now.
